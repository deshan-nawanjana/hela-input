# Sinhala Text Input and Converter

Typing Sinhala text can be different from one platform to another. While a computer can only identify the Unicode Sinhala characters, for designing purposes, using font files can display characters in various styles. Due to these differences, this typing tool can help you to convert those Sinhala texts from one format to another.

Included source files for the converter:

- [ASCII_UNI.json](./assets/objects/converters/ASCII_UNI.json)
- [SIN_UNI.json](./assets/objects/converters/SIN_UNI.json)
- [UNI_ASCII.json](./assets/objects/converters/UNI_ASCII.json)

[Hela.js](./assets/modules/Hela.js) module simply loads the above source files and replaces with given converter mappings.

You can also find the suggestions for sinhala language in [suggestions.json](./assets/objects/suggestions.json) file.
